====================
1. 변수 선언 및 할당
====================
- C++은 Python과 달리 변수를 선언할 때 데이터 타입을 명시해야한다.
- 기본적인 데이터 타입은 다음과 같다.
.. list-table::
    :header-rows: 1
    
    * - int
      - char
      - double
      - float
    * - 정수
      - 문자열
      - 소수10자리
      - 소수7자리            

- 데이터 타입에 unsigned, short, long 등을 추가하여 나타내는 범위를 변경할 수도 있다.
- 기본적으로 signed로 되어있으며 절댓값의 범위를 가진다.

- 아래와 같이 선언과 할당을 할 수 있고 선언과 할당을 동시에 할 수도 있다.
.. code::

    int x,y;
    x = 10;
    y = 20;
    char ch = 'character';

- 데이터 타입끼리 변환도 가능하다.
.. code::

    int x;
    float(x);
